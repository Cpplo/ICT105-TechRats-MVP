## IP Summary

The Student Internship & Entry-Level Job Platform is primarily composed of original work created by the Tech Rats team, including the website prototype, documentation, diagrams, screenshots, and project assets.

The project is developed using standard web technologies (HTML5, CSS3, and JavaScript) without relying on proprietary frameworks or commercial UI templates. GitHub is used for version control and team collaboration throughout development.

OpenAI ChatGPT was used as an AI-assisted productivity tool to support brainstorming, documentation writing, technical explanations, and project planning. All AI-generated content was reviewed, verified, and modified by the team before inclusion in the final project.

The project does not intentionally use copyrighted datasets, proprietary APIs, paid templates, or unlicensed third-party assets. Any external resources used are either team-created or available under licenses that permit educational use.
