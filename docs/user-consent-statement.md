# User Consent Statement

This prototype, developed for the ICT105 Student Internship & Entry-Level Job Platform project, is created for academic learning purposes. We will ask users (students and employers) to interact with the prototype, simulate registration, and provide feedback. Any information entered — including sample profile, verification, or job-posting data — will be used only to demonstrate platform workflows and improve the prototype. Users should not enter real sensitive personal information (e.g. real ID numbers, banking details, or real verification documents). Participation is voluntary, and users may stop at any time without penalty.

## Data Use

The data will be used for class project analysis, prototype improvement, and final project reporting. Sample student data (name, university email, student ID, resume) is used only to demonstrate the registration, verification, and job application workflows (FR-03 to FR-06). Sample employer data (company name, email, business registration) is used only to demonstrate company verification and job posting workflows (FR-07, FR-08, FR-30). No data collected during testing is shared publicly or used outside this course project. Verification documents and IDs are visible only to the Admin role during the demo, consistent with the Role-Based Access Control described in `privacy-and-data-protection.md`.
